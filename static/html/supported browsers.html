supported browsers

Luminotes makes use of some advanced browser features, so not all browsers will work for editing your wiki. Supported browsers include:

  • Internet Explorer 6+
  • Firefox 1.5+
  • SeaMonkey 1.0+
  • Iceweasel 1.5+
  • Iceape 1.0+

The following web browsers are known not to work with some or all Luminotes features, at least currently:

  • Safari
  • Opera
  • Konqueror
  • Lynx
